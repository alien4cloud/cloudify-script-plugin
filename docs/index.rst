Cloudify Script Plugin
======================

The script plugin can be used to map node life cycle operations
and workflows to scripts that are included in your blueprint.
Scripts can be written in python, bash, ruby, you name it

The script plugin comes pre-installed
with the default agent packages and is defined in ``types.yaml``.
The source code can be found at the `Github repo link`_.
